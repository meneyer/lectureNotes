Bootstrap Sizing -- this section is showing off how bootstrap grid works

This process helps to re-size content based upon what screen you are looking at the website at.

The CSS style link in the header section was taken from Bootstrap's website

This is a test paragraph, but it should allow you to see boostrap columns in action!

This section is using the "col-md" command. The "md" stands for medium. When viewing on a small device, like a phone, Bootstrap will forgo the grid and turn it into a block behavior.

There is also a small (phones), large (desktops), and extra large (large desktop) option.

To reference the "Responsive Breakpoints" List, check out the bootstrap website.

This is using the "col-7" command. Bootstrap has 12 slots for a row. So this is using 7 no matter what you are viewing it on.

This is using 5 slots. So all 12 are taken up in this row.

This is on the following row as there are no more slots available in the row above. It is also taking up 5 slots.

This is using "col-md-3" It is combining 2 of the options above.

It will get out of the grid behavoir and move to a block style for smaller devices

However, it is still only using 3 of 12 slots per item for medium and above devices.
